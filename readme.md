You must use a 32-bit Linux system to build this project.
```
git clone https://github.com/doozan/NookManager.git

cd NookManager

./build.sh
```


## MacOS Setup

Install `dos2unix`

```
brew install dos2unix
```

Install `gcc`

```
brew install gcc
```


Clone this repo

```
git clone https://github.com/doozan/NookManager.git
cd NookManager
```

Run build script

```
./build.sh
```
